Laftel Plus

Discover new anime on Laftel more easily.

About This App

Laftel Plus is a Chrome browser extension designed for the Laftel (laftel.net) anime streaming service. It offers a dedicated search page with advanced filtering options such as genre, tags, release year, and airing status. The extension automatically excludes anime you've already rated or marked as "not interested," helping you efficiently discover new titles that match your preferences. Users can also quickly toggle "want to watch" or "not interested" status directly from each anime card without leaving the search page.

Google User Data Usage

Laftel Plus offers an optional cloud sync feature powered by Google Drive. By signing in with your Google account, your "not interested" list can be synced across multiple devices automatically. This feature uses only the drive.appdata scope, which limits access to a hidden app-specific folder in your Google Drive — the extension cannot access any of your other Drive files or documents. The synced data consists of anime IDs, names, and thumbnail URLs only. No data is transmitted to any developer-owned server; all data is stored exclusively in your own Google Drive account. Cloud sync is entirely optional — if you choose not to use it, all data remains stored locally in your browser.

Key Features

  • Dedicated search page with advanced filtering (genre, tags, year, airing status)
  • Auto-exclude rated and not-interested anime from search results
  • Quick action buttons to toggle "want to watch" and "not interested" on each card
  • Infinite scroll for seamless browsing
  • Detail sidebar with anime information
  • Google Drive cloud sync for multi-device support (optional)
