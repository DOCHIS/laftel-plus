라프텔 플러스

최종 수정일: 2026년 4월

1. 개요

라프텔 플러스는 라프텔(laftel.net) 서비스 이용 편의를 위한 도구입니다. 기본적으로 모든 데이터는 사용자의 브라우저에만 저장됩니다. 선택적으로 Google Drive를 통한 클라우드 동기화를 사용할 수 있습니다.

2. 수집하는 정보

본 확장 프로그램은 개인정보를 수집하지 않습니다.

3. 로컬 저장 데이터

다음 데이터는 사용자의 브라우저 로컬 스토리지(chrome.storage.local)에만 저장되며, 외부 서버로 전송되지 않습니다:

  • 라프텔에서 가져온 평가한 작품 목록 (작품 ID, 별점)
  • 라프텔에서 가져온 보고싶다 목록 (작품 ID)
  • 사용자가 지정한 관심없음 목록 (작품 ID)
  • 작품 정보 캐시 (이름, 썸네일 URL, 장르, 매체, 연령등급)
  • 확장 프로그램 설정값 (숨기기 옵션, 필터 설정)

4. 쿠키 사용

라프텔 API 호출 시 사용자의 로그인 상태를 확인하기 위해 라프텔 도메인(laftel.net)의 인증 쿠키를 읽습니다. 이 정보는:

  • 라프텔 API 요청의 인증 헤더에만 사용됩니다
  • 확장 프로그램에 저장되지 않습니다
  • 외부로 전송되지 않습니다

5. 클라우드 동기화 (선택 사항)

사용자가 클라우드 동기화 기능을 활성화하면, 관심없음 목록이 사용자 본인의 Google Drive에 저장됩니다.

  • 저장 위치: 사용자 본인의 Google Drive 앱 데이터 폴더 (사용자에게 보이지 않는 숨김 폴더)
  • 저장 데이터: 관심없음 목록 (작품 ID, 이름, 썸네일 URL)
  • 접근 범위: drive.appdata 스코프만 사용하여, 사용자의 다른 Drive 파일에는 접근하지 않습니다
  • 인증: Chrome 내장 인증 API(chrome.identity)를 통한 Google 계정 인증
  • 개발자 서버에 데이터가 저장되지 않습니다
  • 동기화를 사용하지 않으면 모든 데이터는 브라우저 로컬에만 저장됩니다

6. 데이터 전송

본 확장 프로그램은 다음 서비스로만 데이터를 전송합니다:

  • 라프텔(laftel.net): 평가/보고싶다 목록 조회 및 상태 변경
  • Google Drive (선택 사항): 관심없음 목록 동기화 (사용자 본인의 Drive에만 저장)

7. 제3자 공유

어떠한 데이터도 제3자와 공유, 판매, 전송하지 않습니다.

8. 데이터 삭제

브라우저에서 확장 프로그램을 제거하면 모든 로컬 저장 데이터가 자동으로 삭제됩니다. 또는 브라우저 설정에서 수동으로 삭제할 수 있습니다.

9. 변경사항

본 개인정보처리방침은 변경될 수 있으며, 변경 시 확장 프로그램 업데이트를 통해 공지됩니다.
