Privacy Policy

Laftel Plus

Last updated: April 2026

1. Overview

Laftel Plus is a tool designed to enhance the user experience on the Laftel (laftel.net) streaming service. By default, all data is stored only in the user's browser. Optionally, users can enable cloud sync via Google Drive.

2. Information We Collect

This extension does not collect any personal information.

3. Locally Stored Data

The following data is stored only in the user's browser local storage (chrome.storage.local) and is not transmitted to external servers:

  • Rated anime list (anime ID, rating)
  • "Want to watch" list (anime ID)
  • "Not interested" list (anime ID)
  • Anime info cache (name, thumbnail URL, genre, media type, age rating)
  • Extension settings (hide options, filter settings)

4. Cookie Usage

To verify the user's login status when calling the Laftel API, the extension reads authentication cookies from the laftel.net domain. This information is:

  • Used only in the Authorization header of Laftel API requests
  • Not stored by the extension
  • Not transmitted externally

5. Cloud Sync (Optional)

When the user enables cloud sync, the "not interested" list is stored in the user's own Google Drive.

  • Storage location: User's own Google Drive app data folder (hidden folder not visible to the user)
  • Stored data: "Not interested" list (anime ID, name, thumbnail URL)
  • Access scope: Only drive.appdata scope is used; the extension cannot access any other Drive files
  • Authentication: Google account authentication via Chrome's built-in identity API (chrome.identity)
  • No data is stored on any developer-owned server
  • If sync is not used, all data remains in the browser only

6. Data Transmission

This extension transmits data only to the following services:

  • Laftel (laftel.net): Querying and modifying rated / "want to watch" lists
  • Google Drive (optional): Syncing the "not interested" list (stored in user's own Drive only)

7. Third-Party Sharing

No data is shared, sold, or transmitted to any third party.

8. Data Deletion

Removing the extension from the browser automatically deletes all locally stored data. Data can also be manually deleted through browser settings.

9. Changes

This privacy policy may be updated. Changes will be announced through extension updates.

10. Contact

For inquiries, please contact us via GitHub Issues.
