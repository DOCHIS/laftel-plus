# 개인정보처리방침

**라프텔 플러스** (이하 "확장 프로그램")

최종 수정일: 2025년 1월

---

## 1. 개요

라프텔 플러스는 사용자의 개인정보를 수집, 저장, 전송하지 않습니다. 이 확장 프로그램은 라프텔(laftel.net) 서비스 이용 편의를 위한 도구이며, 모든 데이터는 사용자의 브라우저에만 저장됩니다.

## 2. 수집하는 정보

본 확장 프로그램은 개인정보를 수집하지 않습니다.

## 3. 로컬 저장 데이터

다음 데이터는 사용자의 브라우저 로컬 스토리지(chrome.storage.local)에만 저장되며, 외부 서버로 전송되지 않습니다:

- 라프텔에서 가져온 평가한 작품 목록 (작품 ID, 별점)
- 라프텔에서 가져온 보고싶다 목록 (작품 ID)
- 사용자가 지정한 관심없음 목록 (작품 ID)
- 작품 정보 캐시 (이름, 썸네일 URL, 장르, 매체, 연령등급)
- 확장 프로그램 설정값 (숨기기 옵션, 필터 설정)

## 4. 쿠키 사용

라프텔 API 호출 시 사용자의 로그인 상태를 확인하기 위해 라프텔 도메인(laftel.net)의 인증 쿠키를 읽습니다. 이 정보는:

- 라프텔 API 요청의 인증 헤더에만 사용됩니다
- 확장 프로그램에 저장되지 않습니다
- 외부로 전송되지 않습니다

## 5. 데이터 전송

본 확장 프로그램은 라프텔(laftel.net) 이외의 외부 서버로 데이터를 전송하지 않습니다. 라프텔 API 호출은 사용자의 평가/보고싶다 목록을 가져오거나, 상태를 변경하기 위해서만 수행됩니다.

## 6. 제3자 공유

어떠한 데이터도 제3자와 공유, 판매, 전송하지 않습니다.

## 7. 데이터 삭제

브라우저에서 확장 프로그램을 제거하면 모든 로컬 저장 데이터가 자동으로 삭제됩니다. 또는 브라우저 설정에서 확장 프로그램 데이터를 수동으로 삭제할 수 있습니다.

## 8. 변경사항

본 개인정보처리방침은 변경될 수 있으며, 변경 시 확장 프로그램 업데이트를 통해 공지됩니다.
