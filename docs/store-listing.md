# Chrome 웹 스토어 등록 정보

## 기본 정보

- **이름**: 라프텔 플러스
- **버전**: 1.0.0

## 짧은 설명 (132자 이내)

라프텔에서 평가한 작품, 보고싶다, 관심없음 상태를 한눈에 확인하고 빠르게 관리하세요.

## 상세 설명

라프텔 플러스는 라프텔 애니메이션 스트리밍 서비스를 더 편리하게 사용할 수 있도록 도와주는 확장 프로그램입니다.

### 주요 기능

**작품 상태 시각화**
- 평가한 작품: 별점과 함께 오버레이로 표시
- 보고싶다: 분홍색 하이라이트로 표시
- 관심없음: 어둡게 표시하여 구분

**빠른 액션 버튼**
- 작품 카드에 마우스를 올리면 보고싶다/관심없음 버튼이 나타납니다
- 클릭 한 번으로 상태를 바로 변경할 수 있습니다

**관심없음 관리**
- 라프텔에서 제공하지 않는 관심없음 목록을 확장 프로그램에서 관리
- 팝업에서 목록 확인, 검색, 개별 해제 가능

**설정 옵션**
- 관심없는 작품 숨기기 (기본 ON)
- 평가한 작품 숨기기

**동기화**
- 플로팅 버튼으로 언제든 최신 상태 동기화
- 평가 변경 시 자동 증분 동기화

### 사용 방법

1. 확장 프로그램 설치 후 라프텔에 로그인
2. 팝업에서 "수집하기" 버튼을 클릭하여 초기 데이터 수집
3. 라프텔 사이트에서 작품 상태가 자동으로 표시됩니다

### 권한 안내

- **저장소**: 평가/보고싶다/관심없음 데이터 저장
- **쿠키**: 라프텔 로그인 상태 확인용

이 확장 프로그램은 라프텔 공식 서비스가 아닙니다.

## 개인정보 보호 양식

### 전용 목적 설명

라프텔(laftel.net) 애니메이션 스트리밍 서비스에서 사용자가 평가한 작품, 보고싶다 목록, 관심없음 목록을 시각적으로 표시하고 관리할 수 있도록 돕는 확장 프로그램입니다. 작품 카드에 상태를 오버레이로 표시하고, 빠른 액션 버튼을 통해 상태를 변경할 수 있습니다.

### 권한 사용 근거

#### storage 사용 근거

사용자의 평가한 작품 목록, 보고싶다 목록, 관심없음 목록, 작품 정보 캐시, 설정값을 브라우저 로컬 스토리지에 저장하기 위해 필요합니다. 모든 데이터는 사용자의 브라우저에만 저장되며 외부 서버로 전송되지 않습니다.

#### cookies 사용 근거

라프텔 API 호출 시 사용자의 로그인 상태를 확인하기 위해 라프텔 도메인의 인증 쿠키(at_amss-Co)를 읽습니다. 쿠키 내용은 라프텔 API 요청의 Authorization 헤더에만 사용되며, 저장하거나 외부로 전송하지 않습니다.

#### 호스트 권한 사용 근거

laftel.net 및 하위 도메인에서 작품 카드에 상태 오버레이와 액션 버튼을 표시하고, 라프텔 API(api.laftel.net)를 호출하여 사용자의 평가/보고싶다 목록을 가져오기 위해 필요합니다. 다른 웹사이트에서는 동작하지 않습니다.

### 원격 코드 사용

아니요, 원격 코드 권한을 사용하고 있지 않습니다.

모든 JavaScript 코드는 확장 프로그램 패키지 내에 포함되어 있으며, 외부 스크립트를 로드하거나 eval()을 사용하지 않습니다.
