# Chrome 웹 스토어 등록 정보

## 기본 정보

- **이름**: 라프텔 플러스
- **버전**: 1.1.0

## 짧은 설명 (132자 이내)

라프텔에서 새로운 애니메이션을 더 편리하게 찾아보세요. 평가한 작품과 관심없음을 자동으로 제외하고 필터링합니다.

## 상세 설명

라프텔 플러스는 라프텔 애니메이션 스트리밍 서비스에서 새로운 애니메이션을 더 편리하게 찾을 수 있도록 도와주는 확장 프로그램입니다.

### 주요 기능

**전용 검색 페이지**
- 확장 프로그램 아이콘 클릭 시 전용 검색 페이지가 열립니다
- 장르, 태그, 연도, 방영상태 등 다양한 필터 제공
- 무한 스크롤로 편리한 탐색
- 카드 클릭 시 상세 정보 확인

**확장 프로그램 전용 필터**
- 평가한 작품 제외: 이미 평가한 작품을 목록에서 숨김
- 관심없음 제외: 관심없음으로 표시한 작품 숨김 (기본 ON)

**빠른 액션 버튼**
- 작품 카드에서 보고싶다/관심없음 버튼으로 바로 상태 변경
- 클릭 한 번으로 상태를 바로 변경할 수 있습니다

**관심없음 관리**
- 라프텔에서 제공하지 않는 관심없음 목록을 확장 프로그램에서 관리
- 목록 검색, 개별 해제, 백업/복원 가능

**데이터 동기화**
- 라프텔에서 평가한 작품과 보고싶다 목록을 수집
- 수집된 데이터로 필터링에 활용

### 사용 방법

1. 확장 프로그램 설치 후 라프텔에 로그인
2. 툴바의 확장 프로그램 아이콘 클릭 → 검색 페이지 열림
3. "데이터 수집" 버튼을 클릭하여 평가/보고싶다 목록 수집
4. 필터를 설정하고 새로운 애니메이션을 찾아보세요

### 권한 안내

- **저장소**: 평가/보고싶다/관심없음 데이터 및 설정 저장
- **쿠키**: 라프텔 로그인 상태 확인 및 API 인증

이 확장 프로그램은 라프텔 공식 서비스가 아닙니다.

## 개인정보 보호 양식

### 전용 목적 설명

라프텔(laftel.net) 애니메이션 스트리밍 서비스에서 사용자가 평가한 작품, 보고싶다 목록, 관심없음 목록을 기반으로 새로운 애니메이션을 필터링하여 찾을 수 있도록 돕는 확장 프로그램입니다. 전용 검색 페이지에서 다양한 필터와 함께 작품을 탐색하고, 빠른 액션 버튼으로 상태를 변경할 수 있습니다.

### 권한 사용 근거

#### storage 사용 근거

사용자의 평가한 작품 목록, 보고싶다 목록, 관심없음 목록, 필터 설정, 작품 정보 캐시를 브라우저 로컬 스토리지에 저장하기 위해 필요합니다. 모든 데이터는 사용자의 브라우저에만 저장되며 외부 서버로 전송되지 않습니다.

#### cookies 사용 근거

라프텔 API 호출 시 사용자의 로그인 상태를 확인하기 위해 라프텔 도메인의 인증 쿠키(at_amss-Co)를 읽습니다. 쿠키 내용은 라프텔 API 요청의 Authorization 헤더에만 사용되며, 저장하거나 외부로 전송하지 않습니다.

#### 호스트 권한 사용 근거

라프텔 API(api.laftel.net)를 호출하여 사용자의 평가/보고싶다 목록을 가져오고, 작품 정보를 조회하기 위해 필요합니다. laftel.net/finder 페이지에서 확장 프로그램 안내 모달을 표시합니다. 다른 웹사이트에서는 동작하지 않습니다.

### 원격 코드 사용

아니요, 원격 코드 권한을 사용하고 있지 않습니다.

모든 JavaScript 코드는 확장 프로그램 패키지 내에 포함되어 있으며, 외부 스크립트를 로드하거나 eval()을 사용하지 않습니다.
